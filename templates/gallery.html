FRM API

Description:

The FRM API is a REST API, meant to serve as a decoupled backend for a Personal Relationship Management application. It was built using NodeJS, Express, PostgreSQL, bCrypt and KnexJS.

FRM

Description:

FRM is a family, friends and acquaintances personal relationship management application. Users can share lists between one another, set notifications and make private lists. It was built using React.

BounceAround

Description:

BounceAround is a trip planning app. A user can set trip dates, and schedule events for days during their trip. To make adding events easier on the user, they can filter by event and location types. It was built using NodeJS, Express, KnexJS, EJS, Bootstrap, PostgreSQL and utilizes the Google Maps API and Google oAuth.

Fridgedize

Description:

Fridgedize allows a user to search for recipes based on what they have available in their refrigerator. It uses HTML, CSS, jQuery and Javascript. It utilizes the Spoonacular API.

BAM Walkthroughs

Description:

BAM Walkthroughs is an ongoing project to help fledgling developers. It offers step-by-step walkthroughs on topics such as building a custom directive in AngularJS to building a completely decoupled app with local authorization and JSON Web Tokens.
